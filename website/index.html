Badger Badger

The Badger Who Badgers You Back to Work .

A lightweight VS Code extension that nudges you back to work when distractions pull you away.

About Badger Badger

Built for developers who want to stay in flow. Badger Badger detects when you alt‑tab away and gently reminds you to return before distractions take over.

Why It Works

Context switching kills productivity. A small nudge at the right moment keeps you in deep work longer and helps you recover faster when your attention drifts.

Features

Alt‑Tab Detection

Knows when VS Code stops being your active window.

Escalating Alerts

Reminders that get harder to ignore the longer you're away.

Customized Intervals

Allows you to choose your intensity.

Installation

Installing Badger Badger manually in VS Code takes less than a minute. Follow these steps to get started:

1. Open the Extensions panel in VS Code

2. Click the ⋯ menu in the top‑right corner

3. Select Install from VSIX…

4. Choose the downloaded .vsix file

5. Reload VS Code if prompted

After installation, Badger Badger will begin monitoring your focus automatically.

FAQ

Does it track me? No — it only checks if VS Code is active.
